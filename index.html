ToDo web application

The ToDo Application is a Spring Boot project that utilizes an H2 database for data storage and manipulation. With Spring Data JPA, it retrieves and passes data between the application and the database. The front-end is designed using JSP, incorporating HTML, CSS, and Bootstrap for a responsive and user-friendly interface. User authentication is implemented through Spring Security, ensuring secure access. Upon logging in, users gain access to their personalized list of todos, enabling them to efficiently manage tasks through features such as adding, editing, deletion, and updating.
